# 🎓 AI Powered Study Companion

A high-performance, aesthetic study management system built with **React 18** and **Tailwind CSS v4**. This application helps students organize subjects, track task progress via a Bento-style dashboard, and generate AI-driven study materials using the **Google Gemini API**.

---

## 🚀 Features

- **📊 Intelligent Dashboard**: Visualizes your study progress using `recharts`.
- **🤖 AI Study Assistant**: Instantly generate summaries, quizzes, and flashcards for any topic.
- **🗂️ Subject Management**: Group your study tasks by subjects with custom color-coding.
- **✅ Task Tracking**: Manage your daily objectives with a "Pending" and "Completed" logic.
- **💾 Data Persistence**: Automatically saves your data to `localStorage` so you never lose progress.
- **✨ Aesthetic UI**: Built with a modern, clean design inspired by high-end productivity tools.

---

## 🛠️ Tech Stack

- **Core**: React 18 (Vite)
- **Styling**: Tailwind CSS v4, Framer Motion (Animations)
- **State**: Context API + useReducer (Global State Management)
- **Icons**: Lucide React
- **API**: Axios + Google Gemini AI (Generative AI)
- **Forms**: React Hook Form + Yup Validation
